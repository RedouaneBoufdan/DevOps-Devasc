Webex API Python Lab (8.6.7)
Doel van dit experiment

In dit lab heb ik geleerd hoe ik de Webex Teams API kan gebruiken met Python.
Het doel is om Webex automatisch te beheren via REST API-calls, zonder de grafische interface te gebruiken.

Gebruikte technologieën

Python 3

Webex Teams REST API

HTTP-methodes: GET en POST

Authenticatie met Personal Access Token

JSON-data

Python library: requests

Wat werd uitgevoerd

In dit experiment heb ik:

mijn Webex-account geauthenticeerd met een access token

mijn gebruikersinformatie opgehaald (/people/me)

een gebruiker opgezocht via e-mail

Webex rooms opgelijst en een nieuwe room aangemaakt

meetinginformatie van een room opgehaald

memberships beheerd (leden van een room)

een Markdown-bericht verstuurd via de Webex API

Resultaat

De scripts werken correct en tonen aan dat Webex Teams kan worden beheerd via Python en REST API’s.
Dit experiment illustreert hoe automatisatie en API-integratie gebruikt kunnen worden binnen een DevOps-context.

Opmerking

Voor dit lab is een geldige Webex Personal Access Token vereist.
Deze token is tijdelijk en moet regelmatig vernieuwd worden.
