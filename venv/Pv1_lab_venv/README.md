# Pv1 – Python Virtual Environment (lab)

Dit experiment is gebaseerd op de slides over Python virtual environments.

## Doel
- Werken met een geïsoleerde Python-omgeving.
- Python packages installeren zonder het systeem te beïnvloeden.
- Begrijpen waarom virtual environments gebruikt worden.

## Stappen

python3 -m venv venv
source venv/bin/activate
pip install requests
pip list
deactivate
